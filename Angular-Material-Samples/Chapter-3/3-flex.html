Flex

 Flex to occupy all available space 
            
        <!-- Following row has two text fields. Both together fill the row -->
        <div layout="row" layout-padding>
            <input flex type="text" placeholder="Text box 1 in a row"></input>
            <input flex type="text" placeholder="Text box 2 in a row"></input>
        </div>
        <!-- Following row has just one text field. Whole row is taken by the text field-->
        <div layout="row" layout-padding>
            <input flex type="text" placeholder="Only text box in a row"></input>
        </div>
        <!-- Here container is a column. Hence second element moves to the next row in the column-->
        <div layout="column" layout-padding>
            <input flex type="text" placeholder="Text field 1 in a column"></input>
            <span>&nbsp;</span>
            <input flex type="text" placeholder="Text field 2 in a column"></input>
        </div>
Define size of each text field using flex 
            
        
        <!--        
            Apply flex="33" results in 100/3. And flex="66" results in 200/3. Two elements with flex="33" and flex ="66", 
            together will take up 100% space. Sum of the two numbers is 100.
        -->       
        <div layout="row" layout-padding>
            <input flex="33" type="text" placeholder="Provide full name"></input>
            <input flex="66" type="text" placeholder="Provide complete address"></input>
        </div>
